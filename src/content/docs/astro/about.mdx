Astro is a framework which can help you make a website. It shines at building static webpages. Websites are static when they do not have dynamic content.

Other popular web frameworks are: React, Vue, Angular, Svelte

## Astro

The pages of this category are about building with Astro. If you are not familiar with Astro I recommend reading the [official documentation](https://astro.build/) first as I will not be covering basics.
